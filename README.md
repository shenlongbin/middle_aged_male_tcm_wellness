# Middle-Aged Male TCM Wellness

This project explores a content-first wellness business for educated middle-aged men in China, using modern health logic to explain sub-health and bridge into TCM-informed养生.

## Project Focus

- Audience: Chinese 75s/80s generation educated middle-aged men
- Problem space: sleep decline, fatigue, metabolic warning signs, stress, aging anxiety, and trust gaps in health content
- Market thesis: this audience rejects low-trust elderly marketing but is open to logic-driven, dignified, long-term wellness guidance
- Initial market: domestic China
