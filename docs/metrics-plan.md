# Metrics Plan

## Primary Validation Hypotheses

1. Educated middle-aged men will strongly resonate with a "science-to-TCM wellness bridge" framing.
2. The audience will prefer logic-driven, low-shame health guidance over traditional elderly-health marketing.
3. A simple content-led landing experience can convert some visitors into subscribers or waitlist users.

## Core Metrics

- Unique visitors
- Scroll depth on homepage
- Click-through rate on topic sections
- Email or waitlist conversion rate
- Return visitor rate
- Percentage of visitors selecting top concern category

## Funnel

1. Visit
2. Read hero and problem section
3. Engage with one topic pillar
4. Click CTA
5. Submit lead form

## Event Suggestions

- `view_homepage`
- `scroll_50`
- `scroll_90`
- `click_topic_sleep`
- `click_topic_metabolic`
- `click_topic_stress`
- `click_topic_tcm`
- `click_join_waitlist`
- `submit_waitlist_success`
- `submit_waitlist_error`

## Qualitative Feedback To Collect

- Which problem felt most personally relevant
- Which phrases felt trustworthy or untrustworthy
- Which topic users want next
- Whether users would join a 7-day program
