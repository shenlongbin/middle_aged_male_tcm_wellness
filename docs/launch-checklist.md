# Launch Checklist

## Before MVP Launch

- Positioning is clear in one sentence
- Homepage explains who the project is for
- Three pillar topics exist
- One lead capture path exists
- Basic analytics are installed
- Mobile layout is readable
- Contact path is visible
- Medical overclaims are removed
- Privacy and data collection note is visible if lead capture is enabled

## Distribution Prep

- 5 to 10 launch content ideas prepared
- One main channel selected
- CTA wording tested
- Feedback questions drafted

## Post-Launch Review

- Review first traffic sources
- Review conversion behavior
- Review qualitative comments
- Decide whether to narrow, expand, or reposition
